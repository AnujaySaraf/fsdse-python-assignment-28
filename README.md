# Write a function to sort a dictionary by its keys.

Define a function that takes a dictionary as a parameter and returns a sorted list of the keys of the dictionary.

_Example:_


**Input:**

`{1: 10, 4: 40, 2: 20, 3: 30}`

**Output:**

`[1, 2, 3, 4]`

<br>
<br>

_**Challenge:**_

Do not use for-loop.
